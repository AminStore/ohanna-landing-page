# 📜 Changelog

All notable changes to the **OHANNA** Egyptian Streetwear e-commerce project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.1.0] - 2026-05-21

### Added
* **Mobile Client Documentation**: Created `ohanna-mobile/README.md` and integrated setup steps, Metro running instructions, troubleshooting guides, and system design flowcharts into central `/docs` folders.
* **Standard Repository Files**: Created `CONTRIBUTING.md`, `SECURITY.md`, `CODE_OF_CONDUCT.md`, `CHANGELOG.md`, and `LICENSE.md` in the root repository folder to fulfill production-grade standards.
* **Vercel Web App Config**: Configured `vercel.json` files for the web frontend storefront (`ohanna/`) and backend Express server (`api-server/`).

### Changed
* **Documentation Architecture**: Consolidated dispersed readme parameters into single-truth guides:
  * `docs/SETUP.md` (Local workspace setups and tools)
  * `docs/DEPLOYMENT.md` (Production configs, EAS builds, cloud hosts)
  * `docs/ARCHITECTURE.md` (Repository system mapping & diagrams)
  * `docs/API.md` (Endpoint schemas, Zod validation, payload references)
* **Root Showroom**: Refactored the root `README.md` into an elegant, visual landing portal featuring quick-navigation grid cards and active milestone tracks.

---

## [1.0.0] - 2026-05-20

### Added
* **Frontend Web App (`ohanna/`)**: Responsive, dark-mode Egyptian streetwear catalog page with animated shopping cart drawers, blur overlay effects, and stripe fallback checkout routines.
* **Backend Express Server (`api-server/`)**: Express API with Swagger UI endpoint descriptions, schema validations via Zod, Stripe session setups, and Pino logging.
* **Autogenerated Sync Layer**: Installed Orval codegen pipeline to compile OpenAPI specifications (`openapi.json`) into type-safe react hooks and shared TypeScript models.
* **Mobile Storefront Core (`ohanna-mobile/`)**: Initialized cross-platform application layout using Expo Router, React Native, custom blurred glassmorphism cards, and React Query hooks.
